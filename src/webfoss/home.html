展示性网站 / E-Promotion Website

展示型网站顾名思义, 主要作用是通过网站来展示企业形象和展示公司产品，同时通过新闻报道，应用解决方案等栏目让访问者了解企业。

电子商务型网站 / E-Commerce Website

电子商务通常是指是在全球各地广泛的商业贸易活动中在因特网开放的网络环境下，基于浏览器/服务器应用方式，买卖双方在线进行各种商贸活动。

手机型网站 / Mobile Site

随着智能手机的普及，手机网站越来越成为国际上一个趋势。手机网站（WAP）的特点是完全根据手机屏幕尺寸和功能度身定制。

微站 / Micro Station

电子商务通常是指是在全球各地广泛的商业贸易活动中在因特网开放的网络环境下，基于浏览器/服务器应用方式，买卖双方在线进行各种商贸活动。

电子邮件 / EDM

EDM/Newsletter也是目前非常主流的一种营销方式，相对传统的方式，更加便捷和经济。而优秀的EDM设计则显得尤为重要。

SEO / SEM

SEO是指搜索引擎优化，SEM是指搜索引擎营销旨，二者旨在在提高搜索引擎中关键词排名，从而获得更多用户访问和流量。
